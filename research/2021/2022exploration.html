Lime USB


Early in the Fall, I introduced Lime USB, a concept that allowed a full Windows operating system to run on a USB stick. We finally did it, now when your PC dies it doesn’t need to stop there. The USB sticks will start at 64 gigabytes for $150, and go all the way up to 256 for $400. Now I understand that the prices might be a little steep, but it is because of the Windows Licensing service.
This brings a lot of new and exiting opportunity for not just common people, but administrators as well, now all of the programs, files, and settings for a user can be taken with them. Schools and Workplaces can request a installation without a product key and use the Microsoft Key Management Services (MKMS) for a smaller bulk price.
